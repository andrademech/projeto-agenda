# Agenda Project MVC

This project was build using Node an MongoDB.

## Deploy

https://projeto-agenda-seven.vercel.app/

## Git clone

On your terminal, run the following command:

### `git clone https://github.com/andrademech/projeto-agenda.git`

## Available Scripts

In the project directory, you can run:

### `npm install`

### `npm start`
